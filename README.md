# Token Price Monitor Telegram Bot

Телеграм-бот для мониторинга цены токена 419INU с автоматическим обновлением данных и расчетом прибыли/убытков для пользователей.

## 🚀 Функциональность

- **Мониторинг цены токена** в реальном времени через DexScreener API
- **Персональные расчеты** баланса, профита и PnL для каждого пользователя
- **Автоматическое обновление** сообщений каждые 6 секунд
- **Кэширование данных** для оптимизации API запросов
- **Умное обновление** - редактирование сообщений только при изменении данных

## 📊 Отображаемая информация

- Текущая цена токена (в USD)
- Текущий баланс пользователя
- Процент профита
- PnL (Profit and Loss)
- 24-часовое изменение цены
- Рыночная капитализация
- Время последнего обновления

## 🛠 Технические детали

### Используемые технологии:
- **aiogram 3.x** - современная асинхронная библиотека для Telegram Bot API
- **asyncio** - асинхронное программирование для обработки множественных запросов
- **requests** - HTTP клиент для работы с DexScreener API
- **datetime** - работа с временными зонами (UTC+5)


## 🔧 Установка и запуск

1. **Клонирование репозитория:**
```bash
git clone https://github.com/soroka01/Token-DexScreener.git
cd Token-DexScreener
```

2. **Установка зависимостей:**
```bash
pip install -r requirements.txt
```

3. **Настройка:**
- Замените `"token"` на реальный токен бота в `main.py`
- Добавьте пользователей в `USER_CONFIG` с их `user_id`, `base_price` и `initial_balance`

4. **Запуск:**
```bash
python main.py
```

## 📝 Использование

1. Найдите бота в Telegram
2. Отправьте команду `/start`
3. Получите актуальную информацию о токене
4. Сообщение будет автоматически обновляться каждые 6 секунд
